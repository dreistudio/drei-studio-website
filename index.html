SnapTag
User Research, Wireframes, UI Design
SnapTag is an iOS app created by DREi Studio. "Snap. Tag. Remember." This visual reminder and to-do list app is designed to help users tag photos and screenshots they take for reference – but later forget about anyways. To refine the initial concept into a design that addresses people's existing pain points, we conducted a user survey to identify different categories of content and current workarounds. A WhatsApp-based diary study helped provide more qualitative data on the moments in which people take or retrieve these "snaps." We defined the product design, detailing user flows with wireframes and initial iOS screen designs, complete with logo and color palette in Figma for the first version to quickly get user feedback. Then we hired a freelance designer for a refined UI refresh.

This One
Product Discovery, User Research, Personas, Design Sprints
We partnered with the team of This One to define and iterate on the functionality of their domain-specific recommender system. Starting with the domain of movie recommendations, we conducted Design Sprints resulting in concepts and wireframes that showcase innovative potential solutions. Based on user interviews and a set of user tests, we derived data-driven Ideal Customer Profiles and supporting Personas. By conducting think-aloud studies of competing products, we also identified categories and severity of pain points in the existing user experience. These UX deliverables informed the prioritization of the product roadmap, driving continuous improvement.
